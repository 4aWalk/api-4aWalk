# api-4aWalk
Projet de développement en Java

Ce projet consiste à réaliser une application mobile permettant de gérer des randonnées.
Au sein d'une randonnée, des produits et des équipements y sont associés et doivent être optimisés afin de démarrer un parcours avec cette randonnée.
